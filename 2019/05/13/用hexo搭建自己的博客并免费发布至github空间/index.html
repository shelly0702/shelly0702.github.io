用hexo搭建自己的博客并免费发布至github空间

具备以下环境

  1. git已安装
  2. node已安装

具体步骤

  1. 电脑上创建放置博客的文件夹

  2. 打开git bash终端，全局安装hexo-cli依赖

  1. 在放置博客的文件夹的位置执行如下命令，此命令的执行会从github上克隆下来博客模板项目并安装上相应的依赖包。

hexo serve

hexo g

deploy:
type: git
repo: https://github.com/username/username.github.io.git
branch: pages

npm install hexo-deployer-git -save

hexo g
hexo d
`
